G2B Gift Card Website - Deployment Instructions

This guide will help you deploy your G2B Gift Card website with a clean, professional URL that runs 24/7 with no "temporary preview" message.

Important: All of these deployment options are completely free and will give you a clean URL like https://g2b.vercel.app or https://g2b-gift-cards.herokuapp.com.

Option 1: One-Click Deployment (Easiest)

Deploy to Vercel (Recommended)

Vercel provides the fastest and easiest deployment with a clean URL:

  1. Click the button below to deploy to Vercel
  2. Create a free Vercel account if you don't have one
  3. Give your project a name like "g2b"
  4. Click "Deploy"

Your website will be available at https://g2b.vercel.app (or similar)

Deploy to Vercel

Deploy to Netlify

Netlify is another excellent option for free hosting:

  1. Click the button below to deploy to Netlify
  2. Create a free Netlify account if you don't have one
  3. Give your site a name like "g2b-cards"
  4. Click "Deploy site"

Your website will be available at https://g2b-cards.netlify.app (or similar)

Deploy to Netlify

Option 2: Command Line Deployment

If you're comfortable with the command line, you can use our deployment script:

  1. Open a terminal/command prompt
  2. Navigate to the g2b directory
  3. Run the deployment script: ./one-click-deploy.sh
  4. Follow the on-screen instructions
